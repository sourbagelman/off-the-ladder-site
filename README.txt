# Off The Ladder — Polish Pack v1

**What’s inside**
- index.html — Homepage with hero, story, and unified purple header
- gallery.html — Gallery grid pulling from Cloudinary tag `otl-gallery`
- contact.html — Clean form with mailto fallback to Dave@otldfw.com
- images/otl-logo.jpg — Logo used in header and hero (rename or replace as needed)

**How to upload to GitHub**
1. In your repo, open each of these files and click **Upload files** or **Add file → Upload**.
2. Upload `index.html`, `gallery.html`, `contact.html`, and the `images/` folder into the repo (images go into `images/`).
3. Commit directly to `main`.
4. Visit your Cloudflare Pages URL (or custom domain) and refresh.
